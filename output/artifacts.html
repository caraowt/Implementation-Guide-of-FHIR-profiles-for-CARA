This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide.

CARA - Application Invitation

An invitation allowed a user to access to an application.
CARA - Healthcare Professional Profile

The user profile of an Healthcare Professional in the CARA Platform.
CARA - Local Profile

The user profile of a local user in the CARA Platform.
CARA - Patient Profile

The user profile of a Patient in the CARA Platform.
SCP - Care Plan

The Care Plan of a Shared Care Plan.
SCP - Care Team

The Care Team of a Shared Care Plan.
SCP - Contact

A contact person of a Shared Care Plan.
SCP - Contextual Data

The Contextual Data resource is used to capture the contextual information of a Shared Care Plan.
SCP - External Observaton

The Observaton of a Shared Care Plan coming from third parties.
SCP - Goal

The Goal of a Shared Care Plan.
SCP - Observaton

The Observaton of a Shared Care Plan.
SCP - Team Member - Practitioner

The Team Member of a Shared Care Plan with the profile Healthcare Professional.
SCP - Team Member - Related Person

The Team Member of a Shared Care Plan with the profile Caregiver.
